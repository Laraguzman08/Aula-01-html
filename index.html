Tag h

As tags h1, h2, h3, h4, h5 e h6 no HTML são usadas para definir os títulos, cabeçalhos e subtítulos da página.

Tag p

A tag p serve para definir e estruturar parágrafos de texto.

Tag a

A tag a serve para criar hiperlinks.

Tag head

A tag head no HTML é usada como contêiner para metadados, configurando informações cruciais sobre a página que não aparecem diretamente no navegador.

Tag body

A tag body serve para conter todo o conteúdo visível de uma página web.

Tag html

A tag html é o componente fundamental de qualquer página web, atuando como a raiz do documento.

Tag img

A tag img tem a função de incorporar imagens diretamente na página.

Tag title

A tag title define o título de um documento, sendo fundamental para exibir o nome da página na aba do navegador, nos resultados de pesquisa e ao compartilhar links. O title fica dentro do head, pois o título precisa ser o primeiro elemento a aparecer ao carregar uma página web.

Tag div

A tag div serve como um contêiner genérico de nível de bloco para agrupar outros elementos, estruturar layouts e aplicar estilos CSS ou manipulações com JavaScript.

Tag span

A tag span no HTML é um contêiner genérico em linha usado para agrupar, estilizar ou manipular partes específicas de texto ou elementos, sem alterar o fluxo do layout.

Tag ul

A tag ul serve para criar listas não ordenadas ou com marcadores, onde a ordem dos itens não é hierárquica ou sequencial.

Tag ol

A tag ol serve para criar listas ordenadas, onde a sequência dos itens é importante e numerada automáticamente. Ela organiza itens usando números, letras ou algarismos romanos.

Tag li

A tag li é usada para definir cada item individual dentro de uma lista, seja ela ordenada ou não ordenada.

Tag br

A tag br no HTML serve para forçar uma quebra de linha no texto, iniciando a próxima parte em uma nova linha sem criar um novo parágrafo.

Tag hr

A tag hr serve para criar uma separação temática ou quebra de assunto entre blocos de conteúdo, geralmente representada por uma linha horizontal. Ela é um elemento de fechamento automático que indica uma mudança de cenário, tópicos ou seções, sendo útil para a semântica da página.

Tag strong

A tag strong é usada para indicar que um trecho de texto possui forte importância, seriedade ou urgência, marcando-o semanticamente para navegadore, leitores de tela e mecanismos de busca, além de aplicá-lo em negrito por padrão.
