Tag h

As tags h1, h2, h3, h4, h5 e h6 no HTML são usadas para definir os títulos, cabeçalhos e subtítulos da página.

Tag p

A tag p serve para definir e estruturar parágrafos de texto.

Tag a

A tag a serve para criar hiperlinks.

Tag head

A tag head no HTML é usada como contêiner para metadados, configurando informações cruciais sobre a página que não aparecem diretamente no navegador.

Tag body

A tag body serve para conter todo o conteúdo visível de uma página web.

Tag html

A tag html é o componente fundamental de qualquer página web, atuando como a raiz do documento.
